Este evento ocurre solo una vez antes de mostrar el formulario de búsqueda. Se puede usar para establecer algunos valores predeterminados en los campos. Por ejemplo: {empresa} = [glo_empresa]; Los campos de fecha se usan de manera diferente. Usan un sufijo "dia" para el día, "mes" para el mes y / o "ano" para el año. Por ejemplo, el campo {Nacimiento} será: {Birth_dia}=date("d"); {Birth_mes}=date("m"); {Birth_ano}=date("Y");
