sc_apl_default - Esta macro permite que el usuario defina en su aplicación inicial lo que sucederá cuando la aplicación pierda la sesión.

sc_language - Esta macro devuelve el idioma de la aplicación.

sc_redir - (Aplicación, Parámetro01; Parámetro02; Destino, Error, height_modal, width_modal)

sc_send_sms - Esta macro se usa para enviar mensajes de texto SMS

sc_send_whatsapp - Esta macro permite el envío dinámico de mensajes de texto a través de Whatsapp.

sc_statistic - Calcula y devuelve una matriz con valores estadísticos, a partir de una matriz con valores numéricos.
