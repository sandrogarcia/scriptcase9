sc_change_css('atributo', 'valor', 'nombre_campo')

Esta macro permite cambiar los atributos CSS de los campos de la aplicación.

 

Lista de parámetros

parámetros valores paramétricos Descripción
nombre_del_atributo

El nombre del botón debe informarse mediante comillas dobles, comillas simples o variables.

Ejemplo
In this example we are using the color attribute

sc_change_css('color', '#dcfce7');

Atributo requerido.

Define el atributo CSS que se cambiará.
valor

El nombre del botón debe informarse mediante comillas dobles, comillas simples o variables.

Ejemplo
En este parámetro definimos el valor para el atributo CSS que debe recibir, en el ejemplo estamos definiendo el color: #dcfce7

sc_change_css('color', '#dcfce7');

Atributo requerido.

Define el valor que recibirá el atributo informado
nombre_del_campo

El nombre del botón debe informarse mediante comillas dobles, comillas simples o variables.

Ejemplo
En este ejemplo estamos definiendo el cambio en el color de fuente del campo fld_status

sc_change_css('color', '#dcfce7', 'status');

Atributo opcional.

Define el campo donde se aplicará el CSS, si no se ingresa ningún valor, el cambio se realizará en todos los campos de la línea.

Ejemplo

Ejemplo de cambio de color de fuente y uso del atributo sc_badge

switch({taskstatus}) {
	case 'COMPLETED':
		sc_change_css('color', '#dcfce7', 'taskstatus');
		sc_change_css('sc_badge', 'green', 'taskstatus');
		break;
		
	case 'IN-PROGRESS':
		sc_change_css('color', '#1e40af', 'taskstatus');
		sc_change_css('sc_badge', 'blue', 'taskstatus');
		break;
		
	case 'NOT-STARTED':
		sc_change_css('color', '#991b1b', 'taskstatus');
		sc_change_css('sc_badge', 'red', 'taskstatus');
		break;
}
