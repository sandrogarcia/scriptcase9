sc_apl_status - Esta macro Activa / Desactiva las aplicaciones a nivel de usuario.

sc_reset_apl_conf - Esta macro elimina todas las modificaciones efectuadas por la macro "sc_apl_conf".

sc_reset_apl_status - Esta macro elimina todas las variables de estado de seguridad de la aplicación.

sc_reset_menu_delete - Esta macro restaura una estructura de elemento de menú. (eliminado por la macro "sc_menu_delete").

sc_reset_menu_disable - Esta macro se usa para habilitar una estructura de elemento de menú. (deshabilitado por la macro "sc_menu_disable").
