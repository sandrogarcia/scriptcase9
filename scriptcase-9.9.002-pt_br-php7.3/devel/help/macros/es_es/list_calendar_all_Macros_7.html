sc_ajax_message - Esta macro permite que la aplicación muestre mensajes personalizados.

sc_block_display - Esta macro muestra / oculta dinámicamente los campos de un bloque específico.

sc_field_color - Esta macro cambia el color de un texto de campo determinado.

sc_field_disabled -

Esta macro se utiliza para bloquear un campo para obtener los datos que se escribirían en él.



sc_field_display - Esta macro muestra dinámicamente o no un campo específico.

sc_field_init_off - Esta macro está diseñada para inhibir los campos de consulta en la carga inicial.

sc_field_readonly - Esta macro dinámicamente establece un atributo de campo de campo a "sólo lectura"

sc_field_style - Esta macro le permite modificar dinámicamente el estilo de campo de Consulta.

sc_format_num - Esta macro se utiliza para dar formato a números numéricos.

sc_form_show - Esta macro muestra dinámicamente o no la forma.

sc_set_focus - Esta macro se usa para establecer el foco en un campo de formulario.

sc_text_style - Esta macro permite modificar dinámicamente el estilo del texto del campo de la Consulta.
