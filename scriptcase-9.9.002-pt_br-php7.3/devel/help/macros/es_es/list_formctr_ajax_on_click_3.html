sc_ajax_javascript - Esta macro permite la ejecución de métodos de JavaScript en eventos de formulario / control

sc_alert - Esta macro muestra una pantalla de mensaje Javascript.

sc_calc_dv - Este macro calcule los dígitos de verificación.

sc_decode - Esta macro devuelve el campo o la variable cifrada a su valor original.

sc_encode - Esta macro devuelve el campo o la variable con el contenido cifrado.

sc_error_exit -

Esta macro interrumpe la ejecución de la aplicación si hay mensajes de error generados por la macro "sc_error_message".


sc_error_message - Esta macro genera un mensaje de error.

sc_get_language - Esta macro devuelve la abreviatura del lenguaje utilizado.

sc_get_regional - Esta macro devuelve la abreviatura de la configuración regional utilizada.

sc_get_theme - Esta macro devuelve el nombre del tema de la aplicación.

sc_include - Esta macro se usa para "incluir" rutinas PHP.

sc_include_library - Esta macro incluye un archivo PHP de una librería en la aplicación.

sc_label - Esta macro se usa para modificar dinámicamente la etiqueta del campo de formulario de cuadrícula.

sc_reset_global - Esta macro elimina las variables de sesión recibidas como parámetro.

sc_send_sms - Esta macro se usa para enviar mensajes de texto SMS

sc_send_whatsapp - Esta macro permite el envío dinámico de mensajes de texto a través de Whatsapp.

sc_set_global -

Esta macro se usa para registrar variables de sesión.


sc_statistic - Calcula y devuelve una matriz con valores estadísticos, a partir de una matriz con valores numéricos.

sc_trunc_num - Esta macro se usa para establecer el número de decimales.

sc_url_library - Esta macro devuelve la ruta de un archivo, dentro de una biblioteca, para ser utilizado en las aplicaciones.

sc_warning - Esta macro activa o desactiva dinámicamente el control de mensajes de advertencia.
