sc_apl_default - This macro allows that the user defines in your initial application what will happen when the application lost the session.

sc_language - This macro returns the application language

sc_redir - This macro is used to redirect to another application or URL.

sc_statistic - Calculates and returns an array with statistical values, from an array with numeric values
