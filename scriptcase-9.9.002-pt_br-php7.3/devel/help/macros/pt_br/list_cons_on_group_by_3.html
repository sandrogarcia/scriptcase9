sc_calc_dv - Esta macro executa o cálculo de dígitos verificadores.

sc_groupby_label - Esta macro altera dinamicamente o label dos campos que são apresentados nas linhas de quebra.

sc_reset_global - Esta macro elimina as váriaveis de sessão recebidas como parâmetro.

sc_send_sms - Esta macro permite o envio dinâmico de mensagem de texto SMS

sc_send_whatsapp - Esta macro permite o envio dinâmico de mensagem de texto através do Whatsapp.

sc_set_global -

Esta macro registra variáveis de sessão.



sc_statistic - Calcula e retorna um array com os valores estatísticos, a partir de uma array com valores numéricos

sc_trunc_num - Esta macro tem como finalidade setar o numero de casas decimais.
