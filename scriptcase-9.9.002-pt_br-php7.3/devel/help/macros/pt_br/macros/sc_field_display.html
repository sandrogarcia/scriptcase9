sc_field_display({Meu_Campo}, on/off)

Permite, dinamicamente, exibir ou não um determinado campo.
Em princípio, todos os campos são exibidos (condição "on").

 

Ex. 1:
if ({tp_cli} == "fisica")
{
sc_field_display({cnpj}, off);
}
else
{
sc_field_display({cpf}, off);
}
