sc_send_whatsapp($arr_settings)

Esta macro recebe um array como parâmetro de configuração que contém a nome da API configurada no Scriptcase, mensagem e o número do destinatário.

Os parâmetros que serão utilizados são:

  • profile: Aqui será informado o nome da API configurada na interface do Scriptcase.
  • message: Neste parâmetro é informado a mensagem que será enviada através da macro.
  • to: Permite informar o número do destino da mensagem.

Exemplo de uso: 

$arr_settings = array(
                 'profile'  => 'ChatAPI',
                 'message'  => 'Test whatsapp',
                 'to' => '558182499699'
             );

sc_send_whatsapp($arr_settings);
